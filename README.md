## Installation

Install dependencies and get going

```bash
  # clone the repository
  git clone https://...

  # install packages
  npm install

  # run tests
  npm run tests

  # create docs
  npm run docs
```

Setup your Clasp ".clasp.json" File in your Project Directory:

```json
{
  "scriptId": <Script ID>,
  "rootDir": <Root Directory of your Project>,
  "parentId": [<IDs of Parent Files>]
}
```
